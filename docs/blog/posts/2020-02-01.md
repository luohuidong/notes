# 斐波那契数列的实现

关于斐波那契数列的介绍，可以直接查看维基百科的[斐波那契数列](https://zh.wikipedia.org/wiki/%E6%96%90%E6%B3%A2%E9%82%A3%E5%A5%91%E6%95%B0%E5%88%97)。

```js
llet fibnacci = n => {
  if (n === 0) {
    return 0
  }
  let a1 = 0, a2 = 1;

  for (let i = 1; i < n; i ++) {
    [a1, a2] = [a2, a1 + a2];
  }

  return a2
}
```
